# Test-Pervolare - Requisitos

- Docker
- docker-compose

# Clonar repositorio

`git clone https://github.com/mseagal/pervolare-front.git`

# Ejecutar instancia de Docker

Ubicados dentro del proyecto ejecutar:

`docker-compose up`

# URL de acceso

Ingresar a:
[http://localhost:4200](http://localhost:4200)
